Quick Start

- Install: `npm ci`
- Develop: `npm run dev`
- Test (unit): `npm run test`
- Test (E2E): `npx playwright test`
- Typecheck: `npm run typecheck`
- Lint/Format: `npm run lint` / `npm run format`
